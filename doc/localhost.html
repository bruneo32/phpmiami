How to use phpmiami with localhost

    Select a tutorial

  • Short, for experienced users.
  • Long and detailed, for the inexperienced users.

Short   tutorial


    First at all, you will need a localhost server with PHP and MySQL, like xampp.
  • Download phpmiami from github and unzip it in ...\xampp\htdocs\phpmiami (or equivalent directory).
  • Start XAMPP (or equivalent). Make sure Apache and MySQL are running.
  • Now open your internet web browser, and look for localhost/phpmiami.

Long   tutorial

Install XAMPP

  • Install  XAMPP  in C:\xampp (or equivalent in Linux or MAC).
  • Open xampp-control.exe, and start Apache and MySQL.

Install PHPMIAMI

  • Open phpmiami on github and download ZIP.
  • Move it to C:\xampp\htdocs
  • Right click phpmiami-main.zip and select extract here.
    Then rename the folder to phpmiami.
  • Now you can delete phpmiami-main.zip

Running

  • Open your favorite web browser.
  • Search localhost/phpmiami.
